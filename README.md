##Fork of the official pkgin git repository made to build on Darwin.

Tested and working on Darwin 13 ( Mac OS X 10.9 Mavericks )

Pkgsrc Dependencies:

- bmake 
- sqlite3
- libnbcompat (included here.)
- libfetch
- libarchive
- pkg_install

Here, we use /usr/pkg on Darwin!

Compile:

`$ bmake`

Install:

`$ sudo bmake install`
